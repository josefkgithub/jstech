You can compile a template with a node module. After intstallation of handlebar on node you can:

c:>handlebars post.handlebars - f post.hbs.js

It looks like the nameing convention mytemplate.handlebars is important.

In this case you will include only post.hbs.js. This contains a function to insert your values and get back the html string.

    Advantages:
  • Smaller handlebar library: handlebars.runtime-v2.0.0.js (20KB) instead of handlebars-v2.0.0.js (99KB)
  • No compilation time in production
